# Basic Operations

## Whiteboard

+ How to create a whiteboard?

   [https://youtu.be/RP_vNMs-wQc](https://youtu.be/RP_vNMs-wQc)

   Click *Whiteboard* at the top of the left sidebar, then click the `+ New Whiteboard` button inside.

   Heptabase whiteboards are hierarchical, and you can place sub-whiteboards inside a whiteboard. To create a sub-whiteboard under the current one, right-click on an empty space and select the *Sub-whiteboard* option, or use the shortcut `W`.

+ How to group objects on a whiteboard together?

   [https://youtu.be/SHBShhvEAAs](https://youtu.be/SHBShhvEAAs)

   There are two common ways:

   1. Multi-select the cards you want to group, right-click and choose *Create Section*, or use the shortcut `Cmd/Ctrl` + `G`.

   2. In the whiteboard toolbar on the left, click the … button and select *Section*, or use the shortcut `G`. Then drag to select the objects you want to group.

+ How to create a mindmap?

   [https://youtu.be/wt2xNZub3So](https://youtu.be/wt2xNZub3So)

   There are two common ways:

   1. In the whiteboard toolbar on the left, click the *Mindmap* button, or use the shortcut `M`.

   2. Right-click on the whiteboard and choose the *Mindmap* option.

+ How to annotate a PDF and place annotations onto the whiteboard?

   [https://youtu.be/JFIcuMPD0Zk](https://youtu.be/JFIcuMPD0Zk)

   1. Drag the PDF file onto the whiteboard.

   2. Click the *Add to side panel* button at the top left of the PDF to display it on the right sidebar.

   3. While reading the PDF, you can highlight by selecting text or using the area highlight tool. Drag these highlights onto the whiteboard to visually organize them.

+ How to add a YouTube video to the whiteboard?

   [https://youtu.be/UAkMfTZqLMU](https://youtu.be/UAkMfTZqLMU)

   Copy the YouTube link and paste it on the whiteboard with `Cmd/Ctrl` + `V`.

## Cards

+ How to see where a card is mentioned or which whiteboards it appears on?

   [https://youtu.be/u90FfGKAOag](https://youtu.be/u90FfGKAOag)

   - If the card is on a whiteboard: right-click a card and select *Show* *Info*.

   - If you opened the card in a tab: scroll to the bottom to see the *Info* section.

+ How to open a card on the right side as a reference?

   [https://youtu.be/DR6Km0ztMws](https://youtu.be/DR6Km0ztMws)

    Common methods:

   - Click the *Card Library* button at the top right (shortcut `Cmd/Ctrl` + `2`), then click the card you want to reference.

   - To add a card from a whiteboard to the right side, click the *Add to side panel* button at the top left of the card.

    Other methods:

   - Use the global search tool (shortcut `Cmd` + `O`) to find a card, then press `Option/Alt` + `Enter` to add it to the right panel.

+ How to place a card on the whiteboard?

   [https://youtu.be/l12NXvnZomo](https://youtu.be/l12NXvnZomo)

    Common methods:

   - Click the *Card Library* button at the top right (shortcut `Cmd` + `2`), then drag the card you want onto the whiteboard.

   - In a card tab, click the … button at the top right and choose *Add to whiteboard*, or use the shortcut `Cmd/Ctrl` + `M`.

    Other methods:

   - Use the global search tool (shortcut `Cmd` + `O`) to find a card, then press `Cmd/Ctrl` + `Shift` + `Enter` to add it to the right.

   - On whiteboard A, select some cards, right-click *Copy*. Then on whiteboard B, right-click *Paste* and choose *Synced pasted card*.

+ How to save a web article as a note card?

   [https://youtu.be/C5Nd1LWKhSE](https://youtu.be/C5Nd1LWKhSE)

   Go to the [download page](https://heptabase.com/download) on the Heptabase website, install the browser extension (Web Clipper), and then you can save any webpage as a note card in one click.

+ How to turn journals or highlights into cards?

   [https://youtu.be/loIxwbAouhY](https://youtu.be/loIxwbAouhY)

   In Heptabase, journals and highlights are cards themselves. You can click *Card Library* in the left sidebar and filter by *Card Type* to include these cards.

+ What is the relationship between the Inbox and cards?

   [https://youtu.be/bb2McfdsjL4](https://youtu.be/bb2McfdsjL4)

    The Inbox is used to manage cards that are *unsorted* or *in progress*.

   1. **Unsorted**: Whenever you create a new card from the mobile app, Card Library, or with the shortcut `Ctrl/Cmd` + `N`, the card will automatically appear in the Inbox. After you add it to a whiteboard, you can click *Mark as done* to indicate it’s been organized.

   2. **In Progress**: If you want to focus on writing a card for a period of time, you can add the card to the Inbox and then mark it as done when finished.

## Tags

+ How to create and add tags?

   [https://youtu.be/zi-9HcvYeNg](https://youtu.be/zi-9HcvYeNg)

    Two ways:

   1. Select a card → right-click → *Manage Tags* (shortcut `Cmd/Ctrl` + `T`). If you enter a tag name that doesn’t exist, it will automatically be created.

   2. Click *Tag* in the left sidebar, then click the `+ Tag` button.

+ How are tags in Heptabase different from other apps?

   [https://youtu.be/MZXhia6CaDw](https://youtu.be/MZXhia6CaDw)

   Tags in Heptabase are a type of *database*. For each tag, you can define property fields. For example, you can have a tag called `#MeetingNotes` with properties like *Date* and *Summary*. Whenever a card is tagged with it, the card automatically gains these editable fields.

   If a tag contains many cards, you can view them in a table to quickly see properties across cards, or use a Kanban board to visualize them.

## Search

+ How to search notes?

   [https://youtu.be/eTHc6NztPYM](https://youtu.be/eTHc6NztPYM)

   Click *Search* at the top of the left sidebar, or use the shortcut `Cmd/Ctrl` + `O`.

+ How to view all whiteboards and cards?

   [https://youtu.be/5Z543bN8G5s](https://youtu.be/5Z543bN8G5s)

   - View whiteboards: click *Whiteboard* in the left sidebar to see all whiteboards.

   - View cards: click *Card Library* in the left sidebar to see all cards.

   Note: Whiteboards, cards, and tags at the bottom of the left sidebar act as *tabs*, similar to browser tabs. Opening one will create a new tab. You can close a tab by clicking the X, or pin it as a bookmark.

## Sharing and Collaboration

+ How to publicly share my whiteboard?

   [https://youtu.be/6xpnf-Mi80Q](https://youtu.be/6xpnf-Mi80Q)

   Click the *Share* button at the top right of the whiteboard, select *Publish*, then click *Create public link*.

+ How to collaborate with others?

   [https://youtu.be/fSA5vG5Uivg](https://youtu.be/fSA5vG5Uivg)

   Click the *Share* button at the top right of the whiteboard, enter the collaborator’s email, and they’ll receive an invitation to register a free account and join your whiteboard.
